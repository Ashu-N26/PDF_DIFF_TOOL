# PDF Diff Tool (One URL)

Features:
- Word-level diff with exact bounding boxes
- Red = changed/removed, Green = added
- Summary panel auto-generated inside merged PDF
- Annotated pages after summary
- Side-by-side browser viewer (first-page preview overlays)
- Single Render service (backend + viewer)

## Local
```bash
npm i
npm start
# open http://localhost:3000
